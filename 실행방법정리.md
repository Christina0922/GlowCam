# 📱 앱 실행 방법 정리 - Android Studio 필요 여부

## 🎯 결론: Android Studio는 **선택사항**입니다!

---

## 방법 1: 실제 Android 기기 사용 (Android Studio 불필요) ⭐ 추천

### 필요한 것
- ✅ 실제 Android 스마트폰
- ✅ USB 케이블
- ✅ Node.js (이미 설치되어 있을 가능성 높음)
- ✅ Java JDK (이미 설치되어 있을 가능성 높음)

### Android Studio는 필요 없습니다!

### 실행 단계

1. **기기에서 개발자 옵션 활성화**
   - 설정 → 휴대전화 정보 → 빌드 번호 7번 연속 탭

2. **USB 디버깅 활성화**
   - 설정 → 개발자 옵션 → USB 디버깅 ON

3. **USB로 PC 연결**
   - 기기에서 "USB 디버깅 허용" 팝업 확인

4. **adb 설치 (Android SDK Platform-Tools만 필요)**
   - Android Studio 전체 설치 없이 **Platform-Tools만** 다운로드 가능
   - https://developer.android.com/tools/releases/platform-tools
   - 또는 Android Studio 설치 시 Platform-Tools만 선택

5. **프로젝트 실행**
   ```bash
   cd D:\아이돌\idol-filter-app
   npm install
   npm start
   # 새 터미널에서
   npm run android
   ```

## 방법 2: Android 에뮬레이터 사용 (Android Studio 필요)

### 필요한 것
- ✅ Android Studio (에뮬레이터 사용을 위해 필요)
- ✅ Node.js
- ✅ Java JDK

### 실행 단계

1. **Android Studio 설치**
   - https://developer.android.com/studio
   - 설치 시 "Android SDK", "Android Emulator" 체크

2. **에뮬레이터 생성**
   - Android Studio → Tools → Device Manager
   - Create Device → 기기 선택

3. **에뮬레이터 실행**
   - Device Manager에서 ▶️ 버튼 클릭

4. **프로젝트 실행**
   ```bash
   cd D:\아이돌\idol-filter-app
   npm install
   npm start
   npm run android
   ```

## 방법 3: Expo Go 사용 (가장 간단하지만 제한적)

### 주의: 현재 프로젝트는 네이티브 모듈 사용으로 Expo Go에서 일부 기능 제한

### 필요한 것
- ✅ Node.js
- ✅ Expo Go 앱 (스마트폰에 설치)
- ❌ Android Studio 불필요
- ❌ Java 불필요

### 실행 단계

```bash
# Expo CLI 설치
npm install -g expo-cli

# 프로젝트에서
cd D:\아이돌\idol-filter-app
npx expo start

# QR 코드 스캔하여 Expo Go 앱으로 실행
```

**단점**: react-native-vision-camera 같은 네이티브 모듈은 작동하지 않을 수 있음

## 🎯 추천 방법

### 실제 Android 기기가 있다면 → 방법 1 (실제 기기) 추천!

**이유:**
- ✅ Android Studio 설치 불필요
- ✅ 더 빠름 (에뮬레이터보다)
- ✅ 실제 환경에서 테스트 가능
- ✅ 설정이 간단함

### 실제 기기가 없다면 → 방법 2 (에뮬레이터)

**이유:**
- ✅ 실제 기기 불필요
- ✅ 다양한 기기 테스트 가능
- ❌ Android Studio 설치 필요
- ❌ 초기 설정이 복잡함

## 💡 실제 기기 사용 시 필요한 최소 설정

Android Studio 전체를 설치하지 않고도 가능합니다:

1. **Android SDK Platform-Tools만 다운로드**
   - https://developer.android.com/tools/releases/platform-tools
   - 압축 해제 후 환경변수 Path에 추가

2. **또는 Android Studio 설치 시**
   - "Android SDK Platform-Tools"만 체크
   - 나머지는 선택사항

## ❓ 자주 묻는 질문

### Q: Android Studio 없이도 되나요?
**A:** 네! 실제 Android 기기를 사용하면 Android Studio 없이도 가능합니다.

### Q: Platform-Tools만 필요한가요?
**A:** 네, 실제 기기 사용 시 Platform-Tools만 있으면 됩니다. 전체 Android Studio는 에뮬레이터 사용 시에만 필요합니다.

### Q: 가장 쉬운 방법은?
**A:** 실제 Android 기기가 있다면 USB 연결 방법이 가장 쉽습니다.
