# 액세스 거부 오류 최종 해결 방법

## 🎯 가장 중요한 것

**TempProject는 무시하고 올바른 프로젝트만 사용하세요!**

올바른 프로젝트 경로:
```
D:\GlowCam\idol-filter-app\android
```

---

## ✅ 해결 방법 (우선순위 순)

### 방법 1: 올바른 프로젝트 열기 (가장 중요!)

1. **Android Studio 실행**
2. **File → Open**
3. **`D:\GlowCam\idol-filter-app\android` 폴더 선택**
   - ⚠️ **TempProject가 아닌 android 폴더를 선택!**
4. **OK 클릭**

**TempProject는 완전히 무시하세요!**

---

### 방법 2: Android Studio 완전 종료 후 삭제

1. **모든 Android Studio 창 닫기**
2. **작업 관리자 열기** (Ctrl + Shift + Esc)
3. **다음 프로세스 종료:**
   - `studio64.exe`
   - `java.exe` (Gradle 관련)
   - `gradle.exe` (있다면)
4. **파일 탐색기에서 `TempProject` 폴더 삭제**
5. **또는 `TempProject삭제.bat` 실행** (관리자 권한으로)

---

### 방법 3: 관리자 권한으로 삭제

1. **명령 프롬프트를 관리자 권한으로 실행**
   - 시작 메뉴 → "cmd" 검색
   - "명령 프롬프트" 우클릭 → "관리자 권한으로 실행"
2. **다음 명령 실행:**
   ```bash
   cd /d D:\GlowCam\idol-filter-app
   rmdir /S /Q TempProject
   ```

---

### 방법 4: 컴퓨터 재시작

가장 확실한 방법:
1. 컴퓨터 재시작
2. 재시작 후 `TempProject` 폴더 삭제

---

## 💡 권장 사항

**TempProject를 삭제할 수 없다면:**
- 그대로 두고 무시
- **올바른 프로젝트(`idol-filter-app\android`)만 사용**
- TempProject는 사용하지 않으므로 문제 없습니다

---

2. **`D:\GlowCam\idol-filter-app\android` 폴더를 열었는지 확인**
3. **TempProject는 완전히 무시**
