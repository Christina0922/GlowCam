# GlowCam 앱 화면 미리보기

GlowCam 앱의 주요 화면을 웹 브라우저에서 미리볼 수 있는 데모 페이지입니다.

## 화면 구성

### 1. 홈 화면 (HomeScreen)
- 검은 배경에 핑크 그라데이션 타이틀
- "카메라 시작" 버튼
- 주요 기능 소개 (5가지 필터, 자동 편집, YouTube Shorts 지원)

### 2. 카메라 화면 (CameraScreen)
- 전체 화면 카메라 프리뷰
- 필터 선택 버튼 (우측 상단)
- 촬영 버튼 (중앙 하단)
- 녹화 버튼 (우측 하단)
- 실시간 필터 효과 적용

### 3. 필터 선택 모달 (FilterSelector)
- 6가지 필터 옵션:
  - 📷 원본
  - 💗 핑크 글로우
  - ✨ 페어리 글리터
  - 👑 공주 소프트
  - ⭐ 아이돌 샤프
  - ❄️ 스노우 페어리
- 필터 강도 조절 슬라이더

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. "카메라 시작" 버튼 클릭
3. 필터 선택 버튼으로 다양한 필터 효과 확인
4. 필터 강도 조절로 효과 강도 변경

## 필터 효과

각 필터는 실제 앱에서 GPU 셰이더로 구현되지만, 웹 데모에서는 CSS로 시각화했습니다:

- **핑크 글로우**: 핑크색 오버레이와 글로우 효과
- **페어리 글리터**: 반짝이는 별 효과 애니메이션
- **공주 소프트**: 부드러운 블러 효과
- **아이돌 샤프**: 대비와 밝기 강화
- **스노우 페어리**: 눈송이 애니메이션 효과

## 참고사항

- 이 데모는 실제 앱의 UI/UX를 시각화한 것입니다
- 실제 카메라 기능은 작동하지 않습니다 (시뮬레이션)
- 모바일 앱의 실제 기능은 React Native로 구현되어 있습니다
